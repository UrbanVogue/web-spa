Видалення облікового запису

Якщо ви запитаєте видалення облікового запису, його буде видалено через 14 днів. Протягом цього часу ви можете увійти до свого облікового запису, щоб знову активувати його та скасувати видалення. Через 14 днів видалення стане незворотнім.
